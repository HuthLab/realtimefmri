Installation
============

Checkout the code from github and install using ``pip``.

.. code-block:: bash
    
    git clone https://github.com/gallantlab/realtimefmri.git
    pip install -e realtimefmri

Uninstall
---------

Uninstall the package using ``pip``

.. code-block:: bash
    
    pip uninstall realtimefmri
